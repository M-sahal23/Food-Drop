## 💡 How It Works

### User Adds Item to Wishlist

```
User clicks heart icon
         ↓
JavaScript sends AJAX request
  (X-Requested-With: XMLHttpRequest)
         ↓
Django view receives request
  (@login_required decorator)
         ↓
Get or create Wishlist record
  (unique_together prevents duplicates)
         ↓
Return JSON response
  {"added": true, "food_name": "Pizza", ...}
         ↓
JavaScript updates UI
  - Fill heart with red
  - Change text to "Saved"
  - Show toast notification
         ↓
No page reload!
```

### User Views Wishlist

```
User clicks "Wishlist" in navbar
         ↓
Django filters items by current user
         ↓
Template displays in professional grid
  - 4 columns on desktop
  - 3 columns on tablet
  - 1-2 columns on mobile
         ↓
User sees price, description, buttons
         ↓
Can move to cart or remove
```
