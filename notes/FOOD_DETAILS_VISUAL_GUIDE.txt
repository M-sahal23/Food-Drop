│ BUTTON INTERACTIONS                                                         │
└────────────────────────────────────────────────────────────────────────────┘

[ADD TO CART BUTTON]
─────────────────────────────────────────────────────────────────────────────
Style:         Red background (#ef4f5f) with white text
Icon:          🛒 Cart icon (left side)
Action:        Submits form with selected quantity
Result:        Item added to cart → Redirects to cart page
Hover:         Color darkens, shadow appears, moves up slightly
Disabled:      N/A (always enabled)

[CUSTOMIZE BUTTON]
─────────────────────────────────────────────────────────────────────────────
Style:         White background with red border (#ef4f5f)
Icon:          🎨 Palette icon (left side)
Action:        Opens customization page for this food
Result:        Redirects to customize page (based on category)
Hover:         Light red background, shadow appears
Disabled:      N/A (always enabled)

[QUANTITY BUTTONS]
─────────────────────────────────────────────────────────────────────────────
Minus Button:  Decreases quantity (minimum 1)
Input Field:   Direct entry of quantity (accepts numbers only)
Plus Button:   Increases quantity (no maximum)
Linked Field:  Updates hidden form field for submission

│ EXAMPLE SCENARIO                                                            │
└────────────────────────────────────────────────────────────────────────────┘

SCENARIO: User wants to order a pizza with custom quantity

STEPS:
1. User on Home page sees Pizza card
2. Clicks on pizza card image/name area
   ↓
3. Redirects to Food Details page showing:
   - Large pizza image
   - "🍕 PIZZA" category tag
   - Name: "Deluxe Veggie Pizza"
   - Rating: "⭐ 5.0"
   - Price: "₹349"
   - Description: "Fresh vegetables with special sauce..."
   - Quantity selector: Currently 1
   
4. User wants 3 pizzas, clicks [+] button twice
   - Counter changes: 1 → 2 → 3
   
5. User clicks red "Add to Cart" button
   - Form submits with quantity=3
   - View processes and adds to cart
   - Page redirects to cart showing item added
   
ALTERNATIVE SCENARIO: User wants to customize pizza

3. After step 3, user clicks "Customize" button
4. Redirects to customization page showing:
   - Pizza image
   - Size options (Regular, Medium, Large)
   - Base options (Maida, Wheat)
   - Sauce options (Garlic, Red Pepper, etc.)
   - Toppings selection (Onions, Peppers, etc.)
   - Quantity selector
   
5. User selects: Size=Large, Base=Wheat, Sauce=Red Pepper, Toppings=Onions+Peppers
6. Sets quantity to 2
7. Clicks "Save to Cart"
8. Customized item added to CustomizedCart
9. Redirects to cart showing customized item
