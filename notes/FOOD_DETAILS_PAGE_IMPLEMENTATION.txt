   • Formula: new quantity = quantity from form + existing quantity
   • Maintains backward compatibility with direct add links

   Code Change:
   ```
   def AddToCart(request, id):
       food = get_object_or_404(FoodItems, id=id)
       
       # Get quantity from request (POST or GET)
       quantity = int(request.POST.get('quantity', request.GET.get('quantity', 1)))

       cart_item, created = Cart.objects.get_or_create(
           food=food,
           defaults={
               'quantity': quantity,
               'total_price': quantity * food.price
           }
       )

       if not created:
           # If item already exists, add the new quantity
           cart_item.quantity += quantity
           cart_item.total_price = cart_item.quantity * food.price
           cart_item.save()

       return redirect('cart')
   ```

ROUTING:
  • URL Pattern: path('<int:id>', views.Food_Details, name='fooddetails')
  • Method: GET request to /foods/<food_id>/
  • Returns: fooddetails.html template with fooditem context

BUTTON FUNCTIONALITY:

A. Quantity Controls:
   JavaScript Functions:
   • increaseQty(): Increments quantity counter
   • decreaseQty(): Decrements quantity counter (min: 1)
   • updateQuantityField(): Syncs visible quantity to hidden form field
   • Form submission automatically includes selected quantity

B. Add to Cart Button:
   • POST request to AddToCart view
   • Includes quantity parameter
   • View creates or updates cart item
   • Redirects to cart page

C. Customize Button:
   • GET request to Customize_Food view
   • Shows customization options based on food category
   • Allows selecting: size, base, sauce, toppings
   • Saves customization to CustomizedCart model
