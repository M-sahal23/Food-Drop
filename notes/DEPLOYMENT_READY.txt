================================================================================
                          API ENDPOINTS
================================================================================

1. POST /foods/buy-now/
   • Accepts: customer_name (from modal)
   • Gets all cart items
   • Creates Order and OrderItem records
   • Clears cart
   • Redirects to: /foods/order-confirmation/<order_id>/

2. GET /foods/order-confirmation/<order_id>/
   • Displays: Order details, items, customization, total

3. GET /foods/api/cart-count/
   • Returns: JSON with item count
   • Used by: Real-time cart badge update

STEP 2: Update Admin (if not already done)
────────────────────────────────────────────
File: foods/admin.py

Add this code:

  from foods.models import Order, OrderItem
  
  class OrderItemInline(admin.TabularInline):
      model = OrderItem
      fields = ('food', 'quantity', 'price', 'is_customized', 'customization_details')
      readonly_fields = ('price', 'customization_details')
  
  @admin.register(Order)
  class OrderAdmin(admin.ModelAdmin):
      list_display = ('order_id', 'customer_name', 'total_price', 'status', 'created_at')
      list_filter = ('status', 'created_at')
      search_fields = ('order_id', 'customer_name')
      inlines = [OrderItemInline]
  
  @admin.register(OrderItem)
  class OrderItemAdmin(admin.ModelAdmin):
      list_display = ('order', 'food', 'quantity', 'price', 'is_customized')
      list_filter = ('is_customized', 'order__created_at')
      search_fields = ('order__order_id', 'food__name')
