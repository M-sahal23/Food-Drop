FEATURE 1: Buy Button in Cart
──────────────────────────────
Location: templates/foods/cart.html
Button: "🛍️ Buy Now" (Green, Large)
Trigger: Opens bootstrap modal
Modal Contains:
  - Customer name input field
  - Order summary display
  - Total amount
  - Confirm/Cancel buttons

FEATURE 2: Unique Order ID Generation
──────────────────────────────────────
Format: ORD_YYYYMMDDHHMMSS_XXXXXXXX
Example: ORD_20260128143025_A7F2B3E1
Auto-Generated: Yes (in Order.save())
Unique: Guaranteed (UUID component)
Editable: No (read-only field)

FEATURE 3: Order Processing
────────────────────────────
When user confirms purchase:
✓ Validate customer name
✓ Get all cart items (regular + customized)
✓ Calculate total price
✓ Create Order record
✓ Create OrderItem records
✓ Preserve customization details as JSON
✓ Clear cart items
✓ Redirect to confirmation page

FEATURE 4: Order Data Storage
──────────────────────────────
Order Table Stores:
  - order_id (unique)
  - customer_name
  - total_price
  - status (default PENDING)
  - created_at (auto)
  - updated_at (auto)

OrderItem Table Stores:
  - order (FK)
  - food (FK)
  - quantity
  - price (snapshot at time of order)
  - is_customized
  - customization_details (JSON)

FEATURE 5: Order Confirmation Page
───────────────────────────────────
Displays:
  ✓ Success message

  ✓ Order date/time
  ✓ Order status
  ✓ List of items with quantities
  ✓ Customization details (if any)
  ✓ Total amount (highlighted)
  ✓ Navigation buttons (Continue Shopping, Back to Home)

FEATURE 6: Cart Icon in Navigation
───────────────────────────────────
Location: Navigation bar (templates/base.html)
Icon: Shopping cart (bi-cart3)
Badge: Red circle with item count
Updates: Real-time via JavaScript fetch

FEATURE 7: Admin Order Management

New Tables Created:

Relationships:
──────────────
