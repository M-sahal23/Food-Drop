HOW TO USE (As Customer):
──────────────────────────

1. Customize Page → Sauce Section:
   • Select a sauce radio button
   • Set quantity next to it (1, 2, 3...)
   • Example: Select "Garlic Sauce" with Qty: 2
   → Price will be ₹10 × 2 = ₹20

2. Customize Page → Toppings Section:
   • Check topping checkbox
   • Set quantity next to it
   • Do this for each topping
   • Example: 
     ├─ Cheese ☑ Qty: 1 → ₹20
     ├─ Pepperoni ☑ Qty: 3 → ₹45
     └─ Onions ☑ Qty: 2 → ₹14

3. In Cart:
   • See each topping listed separately with its price
   • All prices clearly spaced and visible
   • Final total includes all quantities

TROUBLESHOOTING:
──────────────────

Q: Quantity inputs not showing?
A: Clear browser cache, refresh page. Check customize.html saved correctly.

Q: Prices calculating wrong?
A: Verify topping is added multiple times in save_customization().
   Check request.POST.getlist('toppings') receives correct IDs.

Q: Spacing looks wrong in cart?
A: Browser might not support flex layout. Check CSS in cart.html.
   Ensure border-bottom and margin-bottom are set on .price-details p

Q: Toppings showing duplicates?
A: This is expected! If user selects Cheese Qty: 2, it shows twice.
   This is correct behavior - each unit shown separately.
